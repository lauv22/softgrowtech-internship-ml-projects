## Project 1: News Article Summarization using Extractive Method
- **File:** Summarize News Articles with ML.ipynb
- **Description:** An extractive summarization tool that uses NLTK to tokenize sentences, remove stopwords, calculate word frequencies, score sentences, and select the top ones as summary.
- **Technologies used:** Python, NLTK (punkt, stopwords), heapq for ranking
- **How to run:** Open the notebook in Jupyter Notebook or Google Colab → Run all cells → Change the `article` variable to test with new text.
- **Key learnings:** Text preprocessing, frequency-based ranking, basic NLP concepts.

## Project 2: Personality Prediction System Through CV Analysis
- **File:** Personality Prediction System Through CV Analysis.ipynb
- **Description:** Extracts text from a PDF resume using pdfplumber, then predicts one dominant Big Five personality trait (e.g., "Openness High") using TF-IDF vectorization + Logistic Regression (trained on synthetic example data).
- **Technologies used:** Python, pdfplumber (PDF extraction), scikit-learn (TF-IDF, LogisticRegression, Pipeline), NLTK/VADER (sentiment)
- **How to run:** Update the `cv_path` variable to point to a resume PDF → Run the notebook cells.
- **Note:** This is a proof-of-concept; real-world version would use a properly labeled dataset.
- **Key learnings:** PDF parsing, text feature extraction, basic supervised ML classification.
